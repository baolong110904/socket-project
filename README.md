# HTTP Web Proxy Server

## How to download and run
From your command line:

Clone the repo
```sh
git clone https://github.com/baolong110904/socket-project.git
```
Go into the repo
```sh
cd socket-project
```
Open VS Code
```sh
code .
```
Run server
```sh
python proxy.py
```
